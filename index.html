Vodafone, Parallel Wireless and Juniper Collaborate on Open RAN

Juniper Networks, Vodafone and Parallel Wireless are testing a multi-vendor RAN Intelligent Controller (RIC) trial for carrier-specific access control use cases.

The Open Radio Access Network (Open RAN) concept is an initiative of the O-RAN Alliance, a non-profit organization founded by the world's largest mobile operators. The mission of the alliance is to develop open requirements, architecture and a family of standards that describe the next generation radio access network (RAN).

The very paradigm of Open RAN is to create a family of open standards that allow telecom operators to move from relying on traditional vendors to a wide range of equipment manufacturers.

The goal of the current development team is to demonstrate the potential for providing open, flexible resource management and mobile data delivery in any software-driven RAN environment.

A trial version of RIC that supports O-RAN interfaces is being tested in Vodafone test labs in Turkey, but there are plans to port it to the test infrastructure.

More detail

The purpose of the trial is to provide more programmable, automated, granular traffic management for each user. Priority is given to access control capabilities tailored to the needs of tenants, that is, allowing operators to personalize services.

Real-time monitoring and "enforcement" of radio resources across the RAN prioritizes critical users such as hospitals and schools using Juniper's rApp/xApp cloud-based software tools. They manage near-real-time network functions as well as open source parallel wireless functions built in the cloud.

XApps and rApps will allow innovation - in the form of RAN control algorithms - to enter radio access networks orders of magnitude faster than today's proprietary systems. Such innovations will help operators create a differentiated network experience that delivers performance tailored to specific service types, user groups or locations.

This approach is designed to manage, optimize, and automatically adapt services and applications using the RAN and is based on real-time data analysis based on native performance.
